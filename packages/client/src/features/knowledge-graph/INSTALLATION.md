# Installation Instructions

## Required Dependencies

To use the Knowledge Graph Access feature, you need to install Apollo Client and GraphQL:

```bash
npm install @apollo/client graphql
```

## Verification

After installation, verify that the packages are installed:

```bash
npm list @apollo/client graphql
```

You should see both packages listed in your dependencies.

## Next Steps

Once installed, the feature is ready to use. See `README.md` for usage examples.
