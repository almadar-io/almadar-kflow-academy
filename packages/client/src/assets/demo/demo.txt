DETAILED EXPLANATION (for those who want more):

THE USER JOURNEY:

Step 1: Creating a Learning Path
The journey starts on the HomePage where users see all their existing learning paths or can create a new one. Clicking "Start a new learning path" opens a modal where users enter:
- Topic name (e.g., "Spanish", "JavaScript", "Machine Learning")
- Optional focus area (e.g., "Conversational Spanish", "Frontend Development")
- Difficulty level (Beginner, Intermediate, Advanced)
- Goal-focused toggle (structures learning around a specific project/goal)

Step 2: First Level Generation
After clicking "Create Learning Path", the ConceptLoader appears showing real-time generation. The AI creates the first level of concepts (typically 5-10 concepts) along with a learning goal. This goal could be a project (if goal-focused is enabled) or a general learning objective. Users watch as concepts are generated and organized.

Step 3: Exploring the Learning Path
Users land on the Concept List view where concepts are organized by levels. Each level shows:
- The learning goal for that level
- All concepts in that level as cards
- Expandable sections to show/hide levels
- A "Level X Final Review" button for completed levels
- A "Load Next Level" button to generate more concepts

Step 4: Concept Details
Clicking any concept opens the Concept Detail Page showing:
- Concept name and description
- Prerequisites (if any) with ability to add/view them
- Generated lesson content (if available)
- Quick Lesson and Detailed Lesson buttons
- Navigation to previous/next concepts
- Option to load next level if at the end of current level

Step 5: Lesson Generation
Users can generate lessons for any concept. Two options:
- Quick Lesson: Concise 2-3 sentence intro, one example, 2-3 practice questions
- Detailed Lesson: Comprehensive explanation with multiple sections, examples, real-world applications, practice exercises

Lessons include:
- Prerequisites section (if applicable)
- Clear explanations
- Code examples with syntax highlighting and copy functionality
- Practice questions with revealable answers
- Matching questions formatted as tables
- Dialogue examples with proper line breaks
- Real-world applications

Step 6: Visual Exploration
Users can switch to visual views:
- Mindmap View: Interactive tree showing all concepts and their relationships. Single-click to select, double-click to navigate. Supports zoom, pan, and mobile touch gestures.
- Radial View: Circular visualization with seed concept at center, layers radiating outward. Shows concept relationships and hierarchy.

Step 7: Level Completion
When users complete a level, they can access the "Level X Final Review" which provides:
- A comprehensive review of all concepts in that level
- Solution-focused content explaining how concepts relate to the learning goal
- Streamed content for real-time generation
- Fullscreen modal for focused review

Step 8: Progressive Learning
Users can load additional levels at any time. Each new level:
- Builds upon previous concepts
- Introduces new concepts that naturally progress
- Includes a new learning goal (different use case if goal-focused)
- Maintains focus on the original learning objective
